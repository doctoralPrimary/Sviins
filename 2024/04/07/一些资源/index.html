# 神器机场

通过机器人可以直接开通试用套餐 - 白嫖神器
500G 流量使用一个月
他们家之前有不限时的套餐，不过现在好像没有了 <sup> 或者节日会开，有兴趣的可以不定期检查下 </sup>

# 找片机器人

发 GIF 给它就可以帮你找出处的机器人
试了下确实有不少可以立即告诉你出处
虽然说网上有很多热心的老司机
但是当你兴致来了的时候，它说不定是你最坚实的后盾呢？

# 草榴社区

这个网站只能说 dddd , 比较老牌的华语成人社区了吧
当然它好像很久没有开放注册了，所以如果想要账号，得有邀请码
不过好在不注册，我们也能浏览大部分区域

# 机场 - xfltd

他们家我主要用来防失联，10 元买 120G 应该还算不错
